# Carl Fernandes — Portfolio

Static portfolio website (HTML/CSS/JS).

## How to publish on GitHub Pages
1. Create a public repo and upload these files.
2. Go to **Settings → Pages** → **Source: Deploy from a branch** → `main` → `/ (root)` → **Save**.
3. Your site will be live at `https://<your-username>.github.io/<repo>/`.
